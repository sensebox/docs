# Installation of Board Support Packages

<div>
	<div class="description">
		To ensure that the Arduino IDE supports your senseBox MCU and allows you to upload programs to it, you need to install two Board Support Packages beforehand. These include the necessary drivers and software to communicate with the processor of the senseBox. The senseBox Board Support Package also includes our senseBox libraries, providing you with all the essential methods for programming the included sensors.
	</div>

	<h2>Add Board Support Package</h2>

	<p>
		Add the following URL in your Arduino IDE under <em>File - Preferences</em> in the field for <em>Additional Board Manager URLs</em>:
	</p>

	{useBoardStore((state) => state.board) === ':edu S2' ? (
		<div>
			<code>https://espressif.github.io/arduino-esp32/package_esp32_index.json</code>
			<div className="flex justify-center mt-4">
				<img src="/img/arduino-bilder/installation/arduino-bsp-esp32s2.png" />
			</div>
		</div>
	) : (
		<div>
			<code>https://github.com/sensebox/senseBoxMCU-core/raw/master/package_sensebox_index.json</code>
			<div className="flex justify-center mt-4">
				<img src="/img/arduino-bilder/installation/arduino-board-support-url.png" />
			</div>
		</div>
	)}

	<p>
		Open the <em>Board Manager</em> by clicking on the <em>Board Manager</em> icon on the left side (see image).
	</p>
	<div className="flex justify-center mb-6">
		<img src="/img/arduino-bilder/installation/arduino-boards.png" />
	</div>

	{useBoardStore((state) => state.board) === ':edu S2' ? (
		<>
			<p>
				Enter "ESP32" in the search bar at the top to find the packages faster and install the board
				 <code>esp32</code> by Espressif Systems!
			</p>
			<div className="flex justify-center">
				<img src="/img/arduino-bilder/installation/esp-32-boards.png" />
			</div>
		</>
	) : (
		<>
			<p>
				Enter "SAMD" in the search bar at the top to find the packages faster and install the board
				 <code>senseBox SAMD Boards</code> by senseBox and <code>Arduino SAMD Boards</code> by Arduino!
			</p>
			<div className="flex justify-center">
				<img src="/img/arduino-bilder/installation/samd_boards.png" />
			</div>
		</>
	)}

	<div className="mt-8">
		<Admonition type="warning" title="Note on Selecting the Board and Port">
			<p>
				Sometimes you need to manually select the board and port in the Arduino IDE.
				Open the drop-down menu in the top bar and select "Select Other Board and Port".
				In the new window, enter "senseBox" in the search bar, select your board, and the port where your senseBox MCU is connected.
			</p>
		</Admonition>
	</div>

	<div className="flex justify-center">
		<img src="/img/arduino-bilder/installation/arduino_port.png" />
	</div>
</div>

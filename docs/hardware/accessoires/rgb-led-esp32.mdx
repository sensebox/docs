# ESP32 RGB-LED

Mit der RGB-LED kannst du verschiedene Farben darstellen. Die RGB-LED besteht aus drei LEDs, die in den Farben Rot, Grün und Blau leuchten. Durch das Mischen dieser Farben können viele verschiedene Farben dargestellt werden.

<ImageWithText size="xl" src={rgbLed} alt="RGB LED" title="RGB-LED" />{' '}

:::success Module onBoard
Die RGB-LED ist auf der MCU S2 verbaut! Du musst hier also nichts weiter
anschließen.
:::

## Programmierung (Arduino)

Dieser Sketch gibt lässt die RGB-LED jede Sekunde in einer zufällige Farbe leuchten.

```cpp
#include <Adafruit_NeoPixel.h>
Adafruit_NeoPixel rgb_led_1

void setup() {
    rgb_led_1.begin();
    rgb_led_1.setBrightness(30);
}

void loop() {
  if (time_startInterval > time_actualInterval + intervalInterval)
    {
        time_actualInterval = millis();
        rgb_led_1.setPixelColor(0,rgb_led_1.Color(random(0, 255), random(0, 255), random(0, 255)));
        rgb_led_1.show();
    }
}
```

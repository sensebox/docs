# Taster

Auf der Vorderseite der Eye befinden sich drei Taster. Neben einem Taster zum Neustarten des Boards ("Rst") und einem Taster ("Boot"), der manchmal für den Wechsel zwischen bestimmten Modi der Eye verwendet wird, gibt es auch einen frei programmierbaren Taster ("Sw").

:::

## Programmierung (Arduino)

Dieser Sketch gibt in der seriellen Schnittstelle aus, ob der Taster gerade gedrückt wird.

```cpp
void setup() {
  Serial.begin(115200);
  while (!Serial);
  Serial.println("bereit...");
  pinMode(PIN_BUTTON, INPUT_PULLUP);
}

void loop() {
  if (digitalRead(PIN_BUTTON) == LOW) {
    Serial.println("Taster gedrückt");
    delay(50);
  }
}
```

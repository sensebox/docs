# Willkommen auf der openSenseMap - Deiner Plattform für freie Umweltdaten.

<details>
  <summary>Wer steckt hinter dem Projekt?</summary>
  
  Die [opensenseMap](https://opensensemap.org/) ist im Jahr 2014 vom Institut für
  Geoinformatik der Universität Münster ins Leben gerufen worden. Seit dem Start
  ist die openSenseMap eng mit dem Projekt der [senseBox](https://sensebox.de) verbunden.
</details>

<details>
  <summary>Welche Art von Daten kann ich hochladen?</summary>
  
  Die openSenseMap versteht nur numerische Messwerte. Texte, Bilder oder binäre Dateien können nicht übertragen werden.
</details>

<details>
  <summary>Wie kann ich Daten hochladen?</summary>
  
  Die Messwerte können über eine Vielzahl verschiedener Geräte hochgeladen werden.
</details>

<details>
  <summary>Wie viele Daten kann ich pro Minute hochladen?</summary>
  
  Um die Datenbank nicht zu überlasten, können 6 Pakete mit Messwerten pro Minute hochgeladen werden. Du solltest daher darauf achten.
</details>

<details>
  <summary>Wo werden die Daten gespeichert?</summary>
  
  Die Daten werden auf dem Rechenzentrum der Uni Münster gespeichert.
</details>

<details>
  <summary>Welche Geräte werden unterstützt?</summary>
  
  Die openSenseMap empfängt, als offene Plattform, Daten von allen Geräten, die ihre Messdaten zur openSenseMap schicken.
</details>

<details>
  <summary>Ich habe keine senseBox kann ich trotzdem Daten hochladen?</summary>
  
  Nicht nur von der senseBox können Daten auf die openSenseMap geladen werden. Die openSenseMap unterstützt auch Messgeräte aus den Citizen Science Projekten [luftdaten.info](https://luftdaten.info/) und [HackAIR](https://www.hackair.eu/). Dank der [REST API](https://docs.opensensemap.org/) können jedoch von fast jedem IoT-Messgeräte Messwerte an die openSenseMap gesendet werden. Weitere Informationen und wie du dein Messgerät registrierst, findest du unter [https://docs.sensebox.de/](https://docs.sensebox.de/).
</details>

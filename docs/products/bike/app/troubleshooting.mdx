# Allgemeine Schritte

Wenn beim Sammeln von Daten Probleme auftreten, folgen Sie diesen Schritten:
 - Aktivieren Sie automatische App-Updates, um stets die neueste Version der senseBox:bike-App zu verwenden.
 - Starten Sie die senseBox:bike neu: Drücken Sie den Ein-/Ausschalter, um das Gerät auszuschalten, warten Sie einige Sekunden und schalten Sie es wieder ein.
 - Starten Sie die senseBox:bike-App auf Ihrem Smartphone neu.
 - Starten Sie Ihr Smartphone neu und öffnen Sie die senseBox:bike-App.

Wenn keine dieser Maßnahmen hilft, kontaktieren Sie uns bitte unter [kontakt@reedu.de](mailto:kontakt@reedu.de). Bitte geben Sie bei Support-Anfragen folgende Informationen an:
 - Betriebssystem und Version Ihres Smartphones. Bei Android bitte auch Hersteller und Modell angeben.
 - Die verwendete App-Version (senseBox:bike-App > Einstellungen > Info).
 - Eine genaue Beschreibung des Problems und wann es erstmals auftrat.

# Bekannte Probleme und Lösungen

<details>
    <summary>App kann keine Verbindung zur senseBox herstellen</summary>
    - Stellen Sie sicher, dass die senseBox:bike eingeschaltet und vollständig geladen ist.
    - Prüfen Sie, ob Bluetooth auf Ihrem Smartphone aktiviert ist und die App die erforderlichen Berechtigungen hat.

    Wenn Sie ein iPhone besitzen und die senseBox in der App nicht sehen können, führen Sie bitte die folgenden Schritte aus:
    - Installieren Sie [nRF Connect](https://apps.apple.com/de/app/nrf-connect-for-mobile/id1054362403).
    - Öffnen Sie nRF Connect, suchen Sie die senseBox, verbinden Sie sich mit ihr und trennen Sie die Verbindung wieder.

    Sie sollten die Box anschließend wieder in der senseBox:bike-App sehen können.
</details>
<details>
    <summary>Nicht alle Sensordaten werden aufgezeichnet (es gibt Lücken in den Daten)</summary>
    - Stellen Sie sicher, dass der Energiesparmodus auf Ihrem Smartphone deaktiviert ist.
    - Stellen Sie sicher, dass nur die senseBox:bike via Bluetooth mit Ihrem Smartphone verbunden ist.
    - Starten Sie die Aufzeichnung, fahren Sie eine kurze Strecke und stoppen Sie dann.
    - Überprüfen Sie, ob Sie für alle Sensoren (z. B. Geschwindigkeit und Temperatur) die gleiche Anzahl an Datenpunkten haben.
</details>
<details>
    <summary>Die App zeichnet den Track lokal auf, lädt ihn aber nicht auf openSenseMap hoch</summary>
    Workaround: Sie können die aufgezeichnete Strecke als CSV-Datei im openSenseMap-kompatiblen Format exportieren und manuell auf der [openSenseMap-Website](https://opensensemap.org/) hochladen.

    - Starten Sie die App neu und prüfen Sie, ob Echtzeit-Sensordaten angezeigt werden.
    - Stellen Sie sicher, dass Ihre Internetverbindung stabil ist und sich die senseBox:bike in Reichweite Ihres Smartphones befindet.
    - Deaktivieren Sie den Energiesparmodus auf Ihrem Smartphone.
    - Stellen Sie sicher, dass Sie in der App mit Ihrem openSenseMap-Konto angemeldet sind und eine senseBox ausgewählt ist.
    - Starten Sie die Aufzeichnung und fahren Sie einige Meter.
    - Melden Sie sich bei openSenseMap an und prüfen Sie, ob die neuen Daten angezeigt werden.
 </details>
